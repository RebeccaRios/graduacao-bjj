# Site de Confirmação - Graduação Jiu Jitsu

Site para confirmação de presença na graduação de jiu jitsu do dia 02 de fevereiro.

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Ou hospede os arquivos em um servidor web (GitHub Pages, Netlify, Vercel, etc.)

## Funcionalidades

- ✅ Seleção de turma (Kids ou Adulto)
- ✅ Formulário dinâmico baseado na turma selecionada
- ✅ Campos para convidados extras com inputs dinâmicos
- ✅ Informações de pagamento PIX
- ✅ Validação de campos obrigatórios

## Como salvar os dados

Atualmente, os dados são apenas exibidos no console do navegador. Para salvar os dados, você tem algumas opções:

### Opção 1: Google Sheets (Recomendado - Mais Fácil)

**Vantagens:**
- Grátis
- Fácil de configurar
- Você pode ver os dados em tempo real
- Não precisa de servidor

**Como fazer:**
1. Crie uma planilha no Google Sheets
2. Crie um script no Google Apps Script que recebe os dados via webhook
3. Eu posso te ajudar a configurar isso depois que você criar a planilha

### Opção 2: Google Forms (Alternativa Simples)

**Vantagens:**
- Muito fácil de usar
- Dados salvos automaticamente no Google Sheets
- Grátis

**Desvantagem:**
- Você perderia o design customizado do site atual

### Opção 3: Backend simples (Node.js/Python)

**Vantagens:**
- Controle total
- Pode salvar em banco de dados ou arquivo

**Desvantagem:**
- Precisa de servidor/hospedagem
